
# RemNote Theme Template

A customizable theme template for RemNote with support for mathematical notation and reference materials.

## Features

- Clean, minimal design
- Math-friendly styling
- Responsive layout
- Easy customization

## Installation

1. Clone or download this repository
2. Copy the theme files to your RemNote themes directory
3. Activate the theme in RemNote settings

## Customization

Edit the CSS files in the `styles/` directory to customize colors, fonts, and layouts.
